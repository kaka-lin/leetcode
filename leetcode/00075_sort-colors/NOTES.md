# Solution

```
Input: [2,0,2,1,1,0]
Output: [0,0,1,1,2,2]

0, 1, 2: 紅(l)、白(c)、藍(r)

紅色在最左邊，白色在中間，藍色在最右邊
=> 遇到 0 往前扔，遇到 2 往後扔

=================================

遇到 2 往後扔，且最右邊指標往左前進
(因為最後一個一定是 2 且已經處理完畢)
l               l
2 0 2 1 1 0  -> 0 0 2 1 1 2
c         r     c       r

遇到 0 往左扔，且紅白指標一起往右前進
l                 l
0 0 2 1 1 2  -> 0 0 2 1 1 2
c       r         c     r


遇到 0 往左扔，且紅白指標一起往右前進
  l                 l
0 0 2 1 1 2  -> 0 0 2 1 1 2
  c     r           c   r


遇到 2 往後扔，且最右邊指標往左前進
    l               l
0 0 2 1 1 2  -> 0 0 1 1 2 2
    c   r           c r


遇到 1 ， 白色(c) 自己往前進
    l               l
0 0 1 1 2 2  -> 0 0 1 1 2 2
    c r             r
                    c

遇到 1 ， 白色(c) 自己往前進
且 c > r ， 處理完畢
    l               l
0 0 1 1 2 2  -> 0 0 1 1 2 2
      r             r
      c               c

```
